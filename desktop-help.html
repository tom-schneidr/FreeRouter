FreeRouter Desktop

FreeRouter is still a local gateway at http://127.0.0.1:8000/v1, but the desktop app starts it without a console and gives you a normal app window for admin work.

Install Shortcuts

Run .\install-desktop.ps1 once from the project folder. It creates Start Menu and Desktop shortcuts that launch the Tauri desktop app (freerouter_desktop.exe).

Daily Use

  • Open FreeRouter from the Start Menu or Desktop shortcut.
  • Close the window to keep the server running in the tray.
  • Use the tray menu to show the window, restart the server, copy the base URL, or quit.

Notes

The app is private/local. It does not install a Windows service, auto-updater, or signed executable.
